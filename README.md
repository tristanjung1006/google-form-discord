### 1) 디스코드 서버 설정>연동으로 들어가서 웹후크를 생성합니다.
### 2) 웹후크 URL 복사를 합니다.

![image](https://github.com/user-attachments/assets/ea1bf98b-7fa2-4fcf-b872-18537a6bf743)

### 3) 생성한 구글 폼 스프레드시트로 들어가서 확장프로그램>App Script를 클릭합니다.

![image](https://github.com/user-attachments/assets/4fab5885-5535-4f6c-9cec-ec421bd185c5)

### 4) 새 코드를 생성해서 google_script.js 코드를 복사 붙여넣기합니다.

![image](https://github.com/user-attachments/assets/0c3eee58-11a0-4abe-92f5-077f5ffdeeca)

### 5) 생성한 코드를 실행하기 위해서 트리거 설정으로 들어갑니다.
### 6) 응답자가 폼을 제출하는 즉시 알림을 보내기 위해서 그림과 같이 설정합니다.
